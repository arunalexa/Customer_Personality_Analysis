## 📊 Model Performance
After verifying the model against the unseen test set, the following metrics were achieved:

| Metric | Score | Description |
| :--- | :--- | :--- |
| **Accuracy** | 89% | Overall correctness of the model. |
| **Precision** | 0.87 | Ability to find only relevant customers. |
| **Recall** | 0.84 | Ability to find all relevant customers. |
| **F1-Score** | 0.85 | The harmonic mean of Precision and Recall. |

> **Note:** These metrics were calculated on a 20% hold-out test set to ensure the model generalizes well to new customer data.
